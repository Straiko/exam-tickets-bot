# 🎓 Exam Tickets Bot — Бот для изучения билетов

Telegram-бот для подготовки к экзамену по программированию (40 билетов).

## 🚀 Как запустить

### 1. Получи токен бота
1. Открой Telegram, найди **@BotFather**
2. Напиши `/newbot`
3. Введи имя бота (например: `Мои Билеты`)
4. Введи username (например: `exam_tickets_study_bot`)
5. Скопируй токен (формат: `123456789:ABCdef...`)

### 2. Запусти бота
```bash
cd /home/root2506/exam-tickets-bot
export TELEGRAM_BOT_TOKEN='ВСТАВЬ_СВОЙ_ТОКЕН_СЮДА'
python3 bot.py
```

## 📱 Возможности бота

| Режим | Описание |
|-------|----------|
| 📚 **Блоки** | Просмотр билетов по 3 блокам |
| 🎲 **Случайный** | Случайный билет |
| 🧠 **Квиз** | Тест с вариантами ответов |
| 🔥 **Марафон** | Пройти все 40 билетов подряд |
| 📊 **Прогресс** | Шкала прогресса + статистика квиза |
| 🔍 **Поиск** | Просто напиши ключевое слово |

## 📋 Команды
- `/start` — Главное меню
- `/help` — Справка
- `/random` — Случайный билет
- `/quiz` — Квиз
- `/progress` — Прогресс
- `/ticket 15` — Открыть билет №15
